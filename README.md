# VulkanEngine
A abstraction of vulkan for my game engine Zafran, it's in development and not really a big deal i just wanna learn c++ and feel proud
# Building and Running
it is only supported for linux(can be for windows if you tweak a bit), and for Building and Running you can run the "run.sh" file and it will run automatically and produce the binary and the required resources to run in the build folder(yes the "build" folder is more like a "release" folder)
